# testing_nestml_neurons

This folder contains scripts and Jupyter notebooks for testing, validating, and analyzing custom NESTML neuron models used in motor control simulations.

## Structure
- Jupyter notebooks for running and analyzing neuron model simulations
- Scripts for plotting and reporting neuron behavior
- Example data and figures
- `report/`: Documentation and analysis results (if present)

## Usage
Use the provided notebooks and scripts to:
- Test compiled neuron models
- Visualize neuron activity and response
- Analyze model performance and properties

Refer to the main `README.md` and `nestml_neurons/README.md` for environment setup and compilation instructions.
